# E-commerce Mobile Project

This project consists of two main parts:

1.  **Backend:** A Node.js (Express) API with MongoDB.
2.  **Frontend:** A React application with TypeScript and Tailwind CSS.

## Getting Started

Follow the instructions in each directory's `README.md` file to set up and run the backend and frontend separately.

### 1. Backend Setup

Navigate to the `backend` directory and follow its `README.md` for instructions.

```bash
cd backend
npm install
# Configure .env
npm run dev
```

### 2. Frontend Setup

Navigate to the `frontend` directory and follow its `README.md` for instructions.

```bash
cd frontend
npm install
# Configure .env
npm start
```
